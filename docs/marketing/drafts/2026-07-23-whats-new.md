## Draft: r/mcp / r/ClaudeAI post

**Title:** canvas-lms-mcp shipped 23 new tools in 6 weeks — quiz accommodations, grade reconciliation, link audit, and more

---

Since June, [canvas-lms-mcp](https://github.com/bruchris/canvas-lms-mcp) has shipped 23 new tool additions across the Canvas LMS API surface. The server is now at 144 tools total. Here's what's actually new and what you can do with it.

**Role-based tool filtering.** Add `--role student`, `--role teacher`, or `--role admin` to the CLI and the server exposes only the tools relevant to that role. Useful when you're building a student-facing agent and don't want 100+ instructor tools clogging the context window.

**Quiz accommodations (Classic and New Quizzes).** `set_student_quiz_accommodation` and `set_student_new_quiz_accommodation` let an AI agent apply extra-time or extra-attempts accommodations for individual students, matching what instructors would normally do in the Canvas UI. `list_student_quiz_accommodations` reads them back for auditing.

**Grade reconciliation.** `explain_grade` shows a student's weighted grade broken down by assignment group — useful for understanding why a final grade is what it is when multiple groups have different weights. `explain_grading_policy` surfaces the late policy, missing policy, and group weight configuration.

**Link audit.** `audit_course_links` scans course pages and assignments for broken links and cross-course links (the kind that silently break when a course is copied). Returns a structured list of the affected content items and their URLs.

**Submissions awaiting grading.** `list_submissions_awaiting_grading` returns all submissions in a course that have been submitted but not yet graded — sorted by submission date. Useful for building a "what needs my attention today" dashboard.

**Course readiness check.** `check_course_setup` reports on common setup gaps: assignments with no due date, modules that aren't published, missing syllabus, etc. Designed for instructors doing a pre-semester audit.

**Assignment date overrides.** `create_assignment_override` and `set_student_assignment_dates` create per-student or per-section due date extensions. Replaces what was previously a manual UI-only workflow for individual accommodations.

**Content exports.** `create_content_export`, `get_content_export`, `list_content_exports` — trigger and track Canvas course package exports (Common Cartridge, QTI, zip). Useful for backup or migration workflows.

**Grade-by-question quiz review.** `get_quiz_question_responses` returns each student's answer per question for a given quiz, along with points earned. Makes it possible to ask "which question did most students get wrong?" and get a real answer.

**Account notifications.** `list_account_notifications` and `view_account_notifications` (interactive MCP Apps widget) expose the system-wide announcements that Canvas admins post to all users.

**Find duplicate files.** `find_duplicate_files` scans a course's file repository for files with identical content under different names — common after course copy-and-paste workflows.

**Student submission feedback.** `get_my_submission_feedback` lets student agents retrieve instructor comments and peer review feedback on their own submissions.
